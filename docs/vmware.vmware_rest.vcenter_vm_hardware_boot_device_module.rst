Synopsis

Requirements

- python >= 3.6
- aiohttp

Examples
--------

.. code-block:: yaml+jinja

      vcenter_vm_info:
        vm: '{{ search_result.value[0].vm }}'
      register: test_vm1_info
    - name: Set a boot device
      vcenter_vm_hardware_boot_device:
        vm: '{{ test_vm1_info.id }}'
        devices:
        - type: CDROM
